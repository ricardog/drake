* Introduction

This document demonstrates Drake's org-babel integration. Execute any code block
with =C-c C-c= to generate plots inline.

* Basic Plotting

** Simple Scatter Plot

The simplest Drake plot in org-mode:

#+BEGIN_SRC drake :file figures/simple-scatter.svg
(drake-plot-scatter :data '(:x [1 2 3 4 5] :y [2 4 6 8 10])
                   :x :x
                   :y :y
                   :title "Simple Scatter")
#+END_SRC

#+RESULTS:
[[file:figures/simple-scatter.svg]]

** Using Built-in Datasets

Drake includes sample datasets. Let's load the iris dataset:

#+BEGIN_SRC emacs-lisp :results silent
(setq iris (drake-load-csv "../datasets/iris.csv.gz"))
#+END_SRC

Now plot it:

#+BEGIN_SRC drake :file figures/iris-scatter.svg
(drake-plot-scatter :data iris
                   :x :sepal_length
                   :y :sepal_width
                   :hue :species
                   :title "Iris Dataset - Sepal Dimensions")
#+END_SRC

* Sessions for Persistent Data

Sessions maintain data across multiple blocks. Perfect for exploratory analysis!

** Load Data in Session

#+BEGIN_SRC drake :session analysis
(setq tips (drake-load-csv "../datasets/tips.csv.gz"))
(message "Loaded %d rows" (length (plist-get tips :total_bill)))
#+END_SRC

#+RESULTS:
: Loaded 244 rows

** Multiple Plots from Same Session

Now we can create multiple plots using the same data:

*** Scatter Plot

#+BEGIN_SRC drake :session analysis :file figures/tips-scatter.svg

* Variable Passing Between Blocks

You can pass data from one block to another using =:var=:

** Prepare Data

#+NAME: my-data
#+BEGIN_SRC emacs-lisp
'(:x [10 20 30 40 50]
  :y [15 35 25 50 45]
  :category ["A" "B" "A" "B" "A"])
#+END_SRC

#+RESULTS: my-data
| :x | [10 20 30 40 50] | :y | [15 35 25 50 45] | :category | [A B A B A] |

** Plot the Data

#+BEGIN_SRC drake :var data=my-data :file figures/var-passing.svg
(drake-plot-scatter :data data
                   :x :x
                   :y :y
                   :hue :category
                   :title "Data from Another Block")
#+END_SRC

#+RESULTS:
[[file:figures/var-passing.svg]]

* Tips & Tricks

** Tip 1: Quick Templates

Type =<drake= and press =TAB= to insert a Drake code block template:
- =<drake= → Empty block
- =<dscatter= → Scatter plot template
- =<dline= → Line plot template
- =<dbar= → Bar plot template

** Tip 2: Update Plots

Press =C-c C-v u= on a code block to re-execute and refresh the inline image.

** Tip 3: Organize Output

Create a figures directory to keep plots organized:

#+BEGIN_SRC bash :results silent
mkdir -p figures
#+END_SRC

Then use =:file figures/plot-name.svg= in all blocks.

** Tip 4: Conditional Execution

Use =:eval never-export= to prevent re-execution during export:

#+BEGIN_SRC drake :file figures/manual-only.svg :eval never-export
(drake-plot-scatter :data iris :x :sepal_length :y :sepal_width)
#+END_SRC

* Appendix: Header Arguments Reference

| Argument    | Description                  | Example              |
|-------------+------------------------------+----------------------|
| =:file=     | Output filename (required)   | =:file plot.svg=     |
| =:session=  | Named session                | =:session analysis=  |
| =:backend=  | Drake backend                | =:backend rust=      |
| =:theme=    | Drake theme                  | =:theme dark=        |
| =:palette=  | Color palette                | =:palette viridis=   |
| =:var=      | Variable from another block  | =:var data=mydata=   |
| =:id=       | Plot ID for links            | =:id fig1=           |
| =:exports=  | What to export               | =:exports results=   |
| =:eval=     | When to evaluate             | =:eval never-export= |
